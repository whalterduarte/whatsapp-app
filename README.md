## Instalação e Execução do Projeto
Este projeto pode ser executado através do seguinte comando:

1. Vá para a pasta *Backend*
1. Entre no diretório com o comando *cd*
1. Depois disso, execute

```bash
npm i
npm run start

OU

node index.js
